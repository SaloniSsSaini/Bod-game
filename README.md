# BODMAS AI Game

An interactive math learning game that teaches BODMAS using step-by-step problem solving.

## Features
- Step-based solving (not direct answers)
- Visual interaction
- Score, lives, combo system
- Animated UI
- Leaderboard system

## Tech Used
HTML, CSS, JavaScript, PHP

## How to Run
1. Open index.html
2. Run PHP server for leaderboard
